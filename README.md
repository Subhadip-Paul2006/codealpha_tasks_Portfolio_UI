# 🎨 Smritih – Premium Interactive Gallery UI

A modern **interactive gallery web application** designed with a premium visual experience.  
This project focuses on **smooth animations, immersive UI, and modern frontend technologies** to create a visually engaging gallery interface.

Built as a **frontend portfolio project** demonstrating advanced UI design, animation, and responsive layout techniques.

---

## 🚀 Tech Stack

- HTML5
- TailwindCSS
- TypeScript
- GSAP (GreenSock Animation Platform)
- Three.js
- Vite
- npm

These technologies are used to create a **high-performance, animated, and responsive frontend interface**.

---

## ✨ Features

- 🎨 Modern premium UI design  
- 🖼 Interactive image gallery  
- ⚡ Smooth animations using GSAP  
- 🌌 3D visual effects using Three.js  
- 📱 Fully responsive layout  
- ⚙ Fast development environment using Vite  
- 🧩 Modular project structure
